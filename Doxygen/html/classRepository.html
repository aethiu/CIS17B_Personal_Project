template<class K , class T >
virtual const T * Repository< K, T >::update_row ( const K & key,
const T & value )
inlinevirtual

Update table row at key with data from value member variables.

Changing a value's key member variable does not change the key in the Repository's private table. If value and key comparison fails, the function exits without attempting to update and returns nullptr. To change a value's key, create a new row with Repository::create_row then delete the old row with Repository::delete_row.
